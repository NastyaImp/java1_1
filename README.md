# Отчёт о тестировании "Базовое приложение"

## приложение воспроизводит ситуацию: пополнение счета vip-клиента на определенную сумму с переменной для хранения итогового значения

<26.10.2021> - <26.10.2021> было проведено функциональное тестирование приложения "Базовое приложение".

На тестирование затрачено: 1 час

В результате тестирования выявлены следующие дефекты:
* https://github.com/NastyaImp/java1_1/issues/1


## Описание процесса тестирования

В процессе тестирования использовались следующие артефакты*:
* баг-репорт


В качестве тестовых данных использовались данные https://github.com/netology-code/javaqa-homeworks/blob/master/intro/MERGED.md:
* текущий баланс счёта клиента, значение - 2_000_000_000 (два миллиарда рублей)
* сумма перевода, значение - 500_000_000 (пятьсот миллионов рублей)
* переменная для хранения итогового значения - 2_500_000_000
 

Тестирование производилось в следующем окружении:
* 64-разрядная ОС
* java 11
